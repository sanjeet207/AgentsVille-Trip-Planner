AgentsVille Trip Planner Project

Files:
- ProjectStarter.ipynb : Main notebook to run the project
- project_lib.py       : Helper library with all functions and tools

Instructions:
1. Make sure Python 3.10+ is installed.
2. Install required packages if needed (pandas, pydantic, json-repair, etc.).
3. Open ProjectStarter.ipynb and run all cells in order.
4. The notebook will generate a complete travel plan for AgentsVille, run all evaluations, and narrate the trip.
